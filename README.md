# 🐞 Bugless — Full Stack Issue Tracker

Bugless is a full-stack issue tracking system designed to simulate real-world debugging workflows.  
It allows developers to track issues, manage their lifecycle, and document investigation steps clearly.

---

## 🚀 Features

- Create and manage issues
- Status workflow:
  - OPEN → INVESTIGATING → IN_PROGRESS → RESOLVED
- Add investigation notes (logs)
- Add root cause analysis
- Add resolution notes
- Activity timeline (auto-tracked)
- Filter issues by status & severity
- Dashboard with statistics

---

## 🧠 Why this project?

This project simulates how real engineering teams track and resolve bugs.  
It focuses on:

- Clean UI/UX
- Real debugging workflow
- Full-stack architecture
- Scalable structure
